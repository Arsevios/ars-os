# Dashboard Specification

Version: 0.1

# Purpose

Dashboard is the home screen.

It answers one question.

"What should I do right now?"

# Widgets

## Daily Progress

Shows:

- Level
- XP
- Daily XP
- Weekly XP
- Streak

---

## Today's Focus

Maximum three tasks.

Only the most important work.

Not a huge task list.

---

## Active Quests

Current missions.

Example

- Learn SQL
- English B2
- Find Internship
- Upload YouTube Video

---
